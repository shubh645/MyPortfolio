Conditional Statements in Python

Conditional Statements are used to perform different actions based on different conditions. Python has several conditional statements:

  • if
  • if-else
  • elif

if statement

If statement ‚Äî Executes a block when a condition is True. 

x = 10
if x > 5:
    print("x is greater than 5")  # runs because condition is True

Explanation: The expression after if is evaluated. If it's True, the indented block runs. Use elif and else to handle other cases.

if else statement

If-else statement ‚Äî Executes one block when condition is True, another when False. 

age = 15
if age >= 18:
    print("You can vote")  # skipped because condition is False
else:
    print("You cannot vote")  # runs because condition was False

Explanation: When the if condition is False, the else block runs. It's useful for handling two mutually exclusive cases.

elif statement

elif statement ‚Äî Tests additional conditions when the preceding if (or elif) is False. Use it to chain multiple mutually exclusive branches. 

num = 0
if num > 0:
    print("Positive")
elif num == 0:
    print("Zero")
else:
    print("Negative")

Explanation: Python evaluates conditions top to bottom. The first condition that evaluates to True runs its block and the rest are skipped. You can have multiple elif clauses to handle several distinct cases.
