Python datatype

Data type is nothing but the type used to store the values of different variables. Datatypes of variables varies with the required byte space.

Datatypes are as follows:

  • Numeric:
    • int (Integer): Whole numbers
      • count = 42
    • float (Float): Decimal numbers
      • price = 19.99
    • complex (Complex): Complex numbers
      • z = 2 + 3j
  • String:
    • str (String): Text data
      • name = "Shubham"
  • Boolean:
    • bool (Boolean): True or False
      • is_active = True
  • List:
    • list (List): Ordered collection of items
      • fruits = ["apple", "banana", "cherry"]
  • Tuple:
    • tuple (Tuple): Immutable ordered collection of items
      • coordinates = (10, 20)
  • Set:
    • set (Set): Unordered collection of unique items
      • unique_numbers = {1, 2, 3, 4, 5}
  • Dictionary:
    • dict (Dictionary): Key-value pairs
      • person = {"name": "Shubham", "age": 25}
  • NoneType:
    • NoneType: Represents the absence of a value
      • value = None
