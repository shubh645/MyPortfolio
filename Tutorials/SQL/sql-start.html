What is SQL?

SQL (Structured Query Language) is used to manage and manipulate relational databases. It helps you retrieve, insert, update, and delete data efficiently.

Basic SQL Syntax

  • SELECT: Retrieve data from a table
    SELECT * FROM students;
  • INSERT: Add new data
    INSERT INTO students (name, age) VALUES ('Amit', 14);
  • UPDATE: Modify existing data
    UPDATE students SET age = 15 WHERE name = 'Amit';
  • DELETE: Remove data
    DELETE FROM students WHERE name = 'Amit';
