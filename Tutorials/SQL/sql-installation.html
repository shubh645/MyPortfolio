Installation

Windows

  • Go to the MySQL Workbench download page.
  • Select Windows as your operating system.
  • Download the installer (.msi file).
  • Run the installer and follow the setup wizard.
  • Choose “Developer Default” to install MySQL Server and Workbench together.
  • Complete the configuration steps and finish installation.
  • Launch MySQL Workbench from the Start Menu.

macOS

  • Visit the MySQL Workbench download page.
  • Select macOS as your platform.
  • Download the .dmg installer.
  • Open the .dmg file and drag MySQL Workbench to the Applications folder.
  • Open MySQL Workbench from Launchpad or Applications.
  • If prompted, allow permissions in System Preferences > Security & Privacy.

Linux (Ubuntu/Debian)

  • Open a terminal window.
  • Run the following commands:
    • sudo apt update
sudo apt install mysql-workbench
  • Once installed, launch MySQL Workbench from the application menu or by typing mysql-workbench in terminal.

Other Linux Distros

  • Use your package manager (e.g., dnf for Fedora, yum for CentOS).
  • Or download the appropriate package from the official site.
